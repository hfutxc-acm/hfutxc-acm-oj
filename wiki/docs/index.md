# 欢迎来到 HFUTXC ACM 知识库

欢迎来到合肥工业大学宣城校区 ACM 协会的官方知识库！这里汇集了关于算法竞赛、平台使用说明以及各类教程资料。

## 快速导航

* **[平台使用指南](guide.md)**：如果你是新生，刚注册本 OJ，请先阅读本指南，了解如何提交代码、查看评测状态。
* **算法教程**：我们为你准备了从基础到进阶的算法讲义（待补充）。
* **题解精选**：针对校赛、周赛的优质题目，我们收录了详细的解答。

## 为什么使用本知识库？

本知识库基于 **Material for MkDocs** 搭建，拥有极佳的阅读体验：

- 🌓 **白天/夜间模式无缝切换**
- 🔍 **毫秒级的全局搜索**
- 💻 **漂亮的代码高亮**
- 🧮 **完美的 LaTeX 数学公式渲染**：例如 $O(n \log n)$ 或是欧拉公式 $e^{i\pi} + 1 = 0$。
