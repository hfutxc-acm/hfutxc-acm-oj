### 中文

由 FastAPI, Vue 3 和 judge0 构建的轻量而现代化的在线评测系统

#### 构建

```sh
cd backend
uv pip install -r requirements.txt
```

```sh
cd backend
uv run uvicorn main:app --reload
```

```sh
cd frontend
npm run dev
```
